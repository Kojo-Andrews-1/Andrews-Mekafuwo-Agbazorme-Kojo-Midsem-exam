1. Digital Attendance System

A console-based digital attendance system to replace paper-based attendance in lecture halls.

2. Project Overview

This Digital Attendance System is designed to modernize lecture hall attendance taking by providing a digital solution that eliminates paper-based processes. The system addresses common issues with traditional attendance methods including lecture disruptions, data misplacement, and difficulties in generating reliable attendance summaries.

3. Features Implemented

4. Week 1: Student Management
- Student registration with index number, full name, and email
- View all registered students in a formatted table
- Search students by index number
- Data persistence using text files

5. Week 2: Session Management
- Create lecture sessions with course code, date, start time, and duration
- Integration with existing student data
- Session management menu system
- File handling for session data

6. Week 3: Attendance Marking & Reporting
- Mark attendance as Present, Absent, or Late
- Display attendance lists for specific sessions
- Generate attendance summary statistics
- Enhanced input validation

7. Week 4: Complete System & Documentation
- Complete file persistence for all data types
- Code refactoring for improved readability
- Comprehensive project documentation
- Final testing and verification
